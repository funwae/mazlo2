# 设置与偏好

## 1. General

* 默认语言（UI + Prompt preference）

* 日期/时间格式

* 主题（浅/暗）

## 2. Provider & Model

* 默认 Provider（OpenAI / z.ai）

* 默认模型（聊天 / 总监模式 / embedding）

## 3. Memory

* 启用/禁用长期记忆（global toggle）

* 是否显示"建议记忆"

* 是否允许自动生成摘要

## 4. Global Mode

* 是否显示 Mazlo Global 入口

* Global Chat 是否允许访问所有项目摘要
