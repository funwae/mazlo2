# 产品愿景与原则

## 1. 愿景

Chat Ultra with Mazlo 是一个：

> **"多空间、多记忆层级的 AI 工作室"**

特点：

* 每个 Room = 一个真实世界的**项目/场景**（比如 ZekeChat 设计室、EarthCloud 架构室）

* 每个 Thread = 该 Room 内的一个具体子任务

* Mazlo 拥有：

  * Room / Thread / Global 四层记忆

  * 专用 Global Chat 模式作为"总监办公室"

目标用户：

* 高度活跃的创作者 / 架构师 / 开发者

* 长期项目与多话题并行的人（你）

## 2. 设计原则

1. **空间优先（Space-first）**

   * 一切围绕 Room/Thread，不是泛泛"聊天框"。

2. **记忆透明（Memory-Visible）**

   * 记住什么 → UI 可见且可编辑。

3. **选择性长记忆（Selective Long-Term Memory）**

   * 只固化对未来有价值的事实/偏好/计划/项目信息。

4. **双模式推理（Room vs Global）**

   * Room：具体任务、快速、相对细节。

   * Global：跨项目、慢思考、简短高层建议。

5. **栈一致性（Stack-Coherent）**

   * 全部核心状态存于 Convex，前端只做视图和命令。
